# M_club
## 简介

使用Node作为开发语言，基于Express，Mongodb搭建的论坛

## 开发环境

* Node.js:  8.4.0
* MongoDB:  3.4.2
* Express:  4.14.1

## 基础设计

* 采用MVC的设计模式，模块化开发
* 初步设计的功能为登录注册，主页面，个人中心三个模块

## 当前阶段

* 初步完成了上述所有功能
* 登录注册界面有了基本的检测，如用户名字符数，密码重复等
* 主页面根据不同的标签页显示不同的主题帖子，对无人回复的帖子单独列出
* 个人中心可以对自己已发布的帖子进行管理，编辑或删除
